react的起源

React 起源于 Facebook 的内部项目，因为该公司对市场上所有 JavaScript MVC 框架，都不满意，就决定自己写一套，用来架设Instagram 的网站。

react工具的意义


  • Factbook——后台硬

  • Virtual-dom——性能高

  • React Native——一次书写，全端可用
